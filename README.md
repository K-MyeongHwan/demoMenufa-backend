# Logi-backend

Test backend server for salesforce integration demo

## Starting guide

Install depencencies

```console
yarn
or
yarn install
```

Run development server

```console
yarn dev
```

Build

```console
yarn build
```

Run the server with built output

```console
yarn start
```

## Dependencies

- Express
- Node-Postgres
- Typescript
- TS-Node
- Nodemon
